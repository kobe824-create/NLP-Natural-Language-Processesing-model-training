# NLP Natural Language Processing Model Training

A comprehensive Natural Language Processing (NLP) project demonstrating language understanding, processing, and translation capabilities with support for multiple languages including Kinyarwanda.

## 📋 Overview

This repository contains an advanced NLP model that extends beyond traditional language processing by incorporating:
- **Slang and Colloquial Language Recognition**: Understanding informal speech patterns and contemporary language usage
- **Deep Vocabulary Processing**: Enhanced vocabulary recognition for nuanced language understanding
- **Multi-language Support**: Kinyarwanda translation capabilities alongside English language processing
- **Data Analysis**: Housing data analysis and machine learning implementations

## 🚀 Latest Updates (May 8, 2026)

The NLP model has been significantly enhanced to:
- Understand and process slang expressions and colloquial language
- Recognize deeper vocabularies and contextual meanings
- Translate English text to **Kinyarwanda** (Rwanda's natural language)

## 📚 Core Components

### main.py
Implements fundamental data structures and algorithms:
- **Sets**: Unique number management with automatic duplicate removal
- **Stack (LIFO)**: Last-In-First-Out data structure using Python lists
- **Queue (FIFO)**: First-In-First-Out implementation using deque for efficient operations

### dataSet.py
Handles dataset operations:
- Loads the Bengaluru Housing Dataset (CSV format)
- Uses Pandas DataFrame for data manipulation
- Provides descriptive statistics for data exploration

### sklearn.py
Machine learning utilities:
- Loads and processes the Iris dataset
- Demonstrates scikit-learn integration
- Prepares data for classification tasks
